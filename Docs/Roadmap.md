# Roadmap

- Native app shell.
- Repo scanner.
- Local execution policy.
- Build diagnostics.
- AI explanations.
- Diff and rollback system.
- Keychain secret references.

- LaunchAgent workflow orchestration.
- SQLite/GRDB persistence and migrations.

- App Store Connect auth and TestFlight upload.
- Signing/provisioning repair planning.

## Phase 3

- Structured Xcode project mutation.
- Screenshot capture/render/review/upload.
- Metadata drafting and ASC diffing.
- IAP/StoreKit/RevenueCat/Adapty/Superwall scaffolding.
- Compliance evidence auditor.
- Mock ASC server and simulator E2E fixtures.

## Phase 4

- Sparkle updater.
- Developer ID packaging/notarization pipeline.
- Team sync as opt-in cloud.
- CI handoff.
- Collaboration and localization workflows.
